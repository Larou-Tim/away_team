Game Instructions!
  1. Start by creating a character name. Upon entering, you will be given 5 random cards to start

  2. You will be allowed to use one card from each class to help boost your skills.

  3. Press the plane button on the Away Mission Card to start the game.

  4. Beat the "Monsters" by clicking on the orange button and having a higher skill level than them. If you are equal or less than them, you have to try to run away by clicking the pink button. If you fail at running away, you die. Refresh the page to start again.

  5. Beat 10 "Monsters" to win!
